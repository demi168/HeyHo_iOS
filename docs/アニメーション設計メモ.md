# アニメーション設計メモ

## トーン

- **派手**: 全画面オーバーレイ、大きな文字、ゲームの必殺技のようなタメ（溜め）
- **テンポよく**: ボタンタップを起点にリズムよく、次々に送信したくなる。**BPM 177**
- **誰に送ったか・誰から届いたかわかる**: 送信先・送信元が視覚的に分かるようにする

## メモ

- BPM 177 → 1拍 ≈ 約 0.34 秒。アニメーションの区切り（出現・タメ・放出など）をこのテンポに合わせると「リズムよく」感じやすい。
- 「タメ」＝ 必殺技の溜めのように、一瞬止めてから一気に放出する動きを入れると派手さが出る。
- 送信時: 誰に送ったか（友だち名やアイコン）をオーバーレイやフィードバックに含めるとよい。
- 受信時（通知・一覧）: 誰から届いたかが一目で分かるようにする。
