プライバシーポリシー

最終更新日: 2026年4月9日

demiflare168（以下「当方」）は、HeyHo（以下「本アプリ」）における個人情報の取扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

本アプリでは、サービス提供のために以下の情報を収集します。

(1) Apple ID 認証情報

  • Apple Sign In を通じて提供される認証トークン
  • ※ Apple ID のメールアドレス・氏名は本アプリでは取得・保存しません

(2) ユーザーが入力する情報

  • 表示名（ニックネーム）: プロフィールに表示される名前
  • アイコンカラー: プロフィールアイコンの色設定

(3) 自動的に収集される情報

  • デバイストークン: プッシュ通知の送信に使用します。他のユーザーからはアクセスできない領域に保存されます。
  • アカウント作成日時: アカウント管理に使用します。

(4) サービス利用に伴い生成される情報

  • 友だちリスト: 招待コードを通じて登録された友だち関係
  • メッセージ履歴: 送受信した Hey / Ho / Let's Go メッセージの記録（送信者ID、受信者ID、メッセージ種別、送信日時）

2. 情報の利用目的

収集した情報は、以下の目的でのみ使用します。

  • 本アプリのサービス提供（メッセージの送受信、友だち管理）
  • プッシュ通知の送信（メッセージ受信の通知）
  • アカウント管理・本人認証
  • サービスの改善・不具合対応

3. 情報の第三者提供

当方は、以下の場合を除き、収集した個人情報を第三者に提供しません。

  • ユーザーの同意がある場合
  • 法令に基づく場合
  • サービス運営に必要な業務委託先（以下参照）

利用する外部サービス

サービス提供元利用目的
Firebase AuthenticationGoogle LLCユーザー認証
Cloud FirestoreGoogle LLCデータ保存
Firebase Cloud MessagingGoogle LLCプッシュ通知

これらのサービスにおけるデータの取扱いについては、Google プライバシーポリシーをご参照ください。

4. 情報の保存期間

  • ユーザーデータは、アカウントが有効な間保存されます。
  • アカウント削除時に、以下のデータをすべて削除します:
    • ユーザープロフィール（表示名、アイコンカラー）
    • 友だちリスト（双方向）
    • 送受信したメッセージ履歴
    • 招待コード
    • デバイストークン

5. アカウントの削除

ユーザーは、本アプリの「マイページ」>「DELETE ACCOUNT」からいつでもアカウントを削除できます。アカウント削除時には、Apple ID との連携解除および本アプリに保存されたすべてのデータの削除が行われます。

6. セキュリティ

当方は、収集した情報の漏洩、紛失、改ざんを防止するために、以下の対策を講じています。

  • Firebase のセキュリティルールによるアクセス制御
  • 通信の暗号化（HTTPS / TLS）
  • デバイストークン等の機密情報は、他のユーザーからアクセスできない領域に保存

7. お子様のプライバシー

本アプリは、13歳未満のお子様を対象としていません。13歳未満の方が個人情報を提供されたことが判明した場合、速やかに削除いたします。

8. プライバシーポリシーの変更

本ポリシーは、必要に応じて変更することがあります。重要な変更がある場合は、本アプリ内またはその他の適切な方法でお知らせします。
